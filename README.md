🌍 TripNova – Explore the World, One Click at a Time.
------------------------------------------------------------------------------------
Live Demo: https://niaj-morshed.github.io/TripNova--Dream-Destination-Booking-Guide/



🚀 Overview:
-------------
TripNova is a modern, high-performance travel landing page designed to inspire wanderlust.
It features a sleek, cinematic design with advanced UI/UX elements, focusing on smooth 
interactivity and immersive visual storytelling.


✨ Key Features:
-----------------
    ** Cinematic Design: Highly visual layout with high-quality imagery to capture the traveler's imagination.  
    ** Advanced Interactivity: Custom UI elements including a ball-shaped cursor, glow effects, and interactive hover states.  
    ** Fully Responsive: Optimized for all screen sizes, from large desktops to mobile devices, with custom touch-responsiveness.  
    ** Dynamic Navigation: Smooth scrolling and interactive navigation links for an effortless user experience.  
    ** Vibrant Aesthetics: Utilizes dark themes with neon accents and modern color palettes for a cutting-edge look.


🛠️ Technologies Used:
----------------------
Core
   * HTML5: Semantic structure for better SEO and accessibility.
   * CSS3: Custom properties (variables), Flexbox, Grid, and advanced animations (keyframe glows, hover effects).
   * JavaScript: Custom cursor logic, interactive card click events, and mobile-specific responsiveness handling.
